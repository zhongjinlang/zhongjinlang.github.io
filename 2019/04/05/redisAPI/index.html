通用命令

  • keys[patten]（查看所有的key）
  • dbsiz（查看所有key的大小）
  • exists key（判断一个key是否存在）
  • del key [key ..] （删除key）
  • expice key seconds（为key设置过期时间）
  • type key（查看key类型）
